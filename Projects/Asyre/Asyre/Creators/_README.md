## 核心理念

### 为什么需要创作者画像？

```
Asyre 引擎 (通用) + 创作者画像 (定制) = 个性化内容生产系统
```

- **通用引擎**: 意图推导、质量评估、平台适配等核心逻辑
- **定制画像**: 每个创作者独特的风格、偏好、禁忌

### 画像哲学

> "不是让 AI 生成你想要的内容，而是让 AI 理解你是什么样的人。"

画像不是参数填表，而是**渐进式了解**:
- 通过自然对话收集信息
- 通过创作过程观察偏好
- 通过反馈数据验证假设

## 使用方式

### 1. 初始化画像

```bash
/asyre profile init
```

系统会开始问你一系列哲学性/创作性问题，逐步构建你的画像。

### 2. 丰富画像

```bash
/asyre profile enrich
```

继续深入了解你，提升画像完整度。

### 3. 查看画像

```bash
/asyre profile view
```

输出当前画像概览卡片。

### 4. 指定创作者创作

```bash
/asyre --creator=[creator_id] [主题]
```

使用特定创作者画像进行创作。

## 画像维度说明

| 维度 | 内容 | 影响 |
|:-----|:-----|:-----|
| **人格底色** | 道德立场、能量类型、风险偏好 | 内容的价值观和能量感 |
| **认知模式** | 信息来源、思维方式、心智模型 | 内容的逻辑结构和分析角度 |
| **审美偏好** | 秩序感、感官偏好、表达风格 | 内容的视觉和语言风格 |
| **内容身份** | 意图倾向、目标受众、独特角度 | 内容的定位和差异化 |
| **表达规则** | 禁止词、偏好表达、结构习惯 | 内容的具体表达方式 |

## 商业化接口

### 为其他创作者定制画像

1. **咨询师流程**:
   - 使用 `_PROFILE_BUILDING_PROTOCOL.json` 中的问题
   - 与创作者进行 1-2 小时深度对话
   - 生成画像 JSON 文件

2. **画像交付物**:
   - `profile.json` - 完整画像配置
   - 风格样本分析报告
   - Asyre 使用说明

3. **定价参考**:
   - 基础画像构建: 1-2小时咨询
   - 深度画像+风格训练: 3-5小时咨询+样本分析

## 默认创作者: Asher

默认使用 `Asher_Source_Profile_v1/01_Base_Attributes/` 中的参数:

- **意图倾向**: 价值观引导(8), 说服(9), 认知颠覆(9), 信息揭示(8)
- **风格调性**: 冷峻黑客风 + 武器化真诚
- **禁止词**: "大家好"、"众所周知"、"相信很多朋友都"
- **结构规则**: 三点法则、3秒内核心信息、问题感激发
