## 🔑 核心概念

### 什么是 Asher Source Profile?

这是一个**活的系统**，而非静态文档库。它包含：

1. **数字分身 (Digital Twin)**
   - 完整复刻 Asher 的思维逻辑、价值观、认知方式
   - 能以"Asher 的方式"处理问题和做出决策
   - 在 Asher 离线时继续工作

2. **思维外骨骼 (Cognitive Exoskeleton)**
   - 放大 Asher 的认知能力（1人→10人产出）
   - 永不遗忘的完美记忆系统
   - 自动化重复性思维劳动

3. **知识晶体化引擎**
   - 将模糊经验转化为可复用的知识模块
   - 通过 13 个 workflows 标准化处理信息
   - 形成 3D 可视化知识图谱

4. **能力放大器**
   - 即插即用的能力包（Modules）
   - 跨领域知识快速调用
   - AI 辅助决策与执行

## 🚀 核心功能

### 1️⃣ 知识管理 (Knowledge Management)

**目标**: 让知识不再遗忘，持续累积、互相关联

**Workflows**:
- `/learn` - 学习新内容 → 标准化知识文档 + 3D 图谱
- `/fact-check` - 事实核查 + 知识增密
- `/synthesize` - 多文档整合升华
- `/sync-graph` - 同步概念到 3D 知识图谱

**输出**:
- 📖 标准化知识文档（`02_Skill_Tree/Modules/`）
- 🌐 3D 知识图谱（可视化概念网络）
- 📊 术语表 & 公式表

### 2️⃣ 对话处理 (Dialogue Processing)

**目标**: 将对话转化为结构化知识资产

**Workflows**:
- `/raw-transcript` - 原始对话记录（最小编辑）
- `/meeting-analysis` - 深度分析（10 部分结构）
- `/distill-discussion` - 提炼共识与分歧

**输出**:
- 📝 原始记录（`00_Archive/Raw_Transcripts/`）
- 📊 结构化分析（`00_Archive/Meeting_Notes/`）
- 🎯 行动方案 & 共识清单

### 3️⃣ 内容创作 (Content Creation)

**目标**: 基于知识库快速生成高质量内容

**Workflows**:
- `/create-script` - 视频脚本创作
- `/expand-book` - 书籍叙事扩展

**输出**:
- 🎬 视频脚本（带反直觉钩子）
- 📚 书式叙事（融入哲学引用）

### 4️⃣ 工具与自动化 (Tools & Automation)

**Workflows**:
- `/export-pdf` - Markdown → PDF（含 Mermaid 渲染）
- `/render-site` - Markdown → HTML 网站
- `/analyze-contract` - 合同风险分析 + 谈判策略
