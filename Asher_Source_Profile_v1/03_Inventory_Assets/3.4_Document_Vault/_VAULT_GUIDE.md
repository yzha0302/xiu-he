# 3.4_Document_Vault Guide

This vault is designed for **On-Demand Retrieval**. The AI Assistant does not read these files automatically to save context, but will search them when you ask.

## ðŸ“‚ Directory Structure

*   **`00_Inbox/`**
    *   *Usage:* Drop new files here if you are unsure where they go. The AI can help categorize them later.
*   **`01_Personal_Identity/`**
    *   *Content:* Passports, Driver Licenses, National IDs, Birth Certificates, Visas, Police Checks (AFP).
*   **`02_Company_Entities/`**
    *   *Usage:* Create a sub-folder for each company (e.g., `X.H. NOCTUER HOLDINGS`).
    *   *Content:* ASIC Keys, Registration Certs, Constitution, Shareholder Agreements.
*   **`03_Financial_Tax/`**
    *   *Content:* Tax Returns (NOA), Bank confirmation letters, Superannuation annual statements. Avoid dumping monthly statements unless necessary.
*   **`04_Legal_Contracts/`**
    *   *Content:* Lease agreements, Employment contracts, Insurance policies.
*   **`05_Medical_Health/`**
    *   *Content:* Vaccination records, Prescriptions, Major medical reports, Health Insurance (OVHC/OSHC).
*   **`06_Education_Qualifications/`**
    *   *Content:* Degree certificates, Transcripts, Graduation records, English Language Results (PTE/IELTS).
*   **`07_Vehicles_Transport/`**
    *   *Content:* Registration (Rego), Insurance, Service logs, Purchase contracts for vehicles.
*   **`08_Residency_Property/`**
    *   **`Residential_Rental_History/`**: Contains address master list and specific rental property portfolios (Leases, Bonds, Condition Reports).
    *   *Usage:* Keep history of all previous addresses here.

## ðŸ· Naming Convention (Best Practice)

For easier searching, try to name files with: `YYYY-MM-DD_Type_Description`

*   *Good:* `2025-06-09_ASIC_CorporateKey_Letter.pdf`
*   *Good:* `2024-07-01_ATO_NoticeOfAssessment.pdf`
*   *Bad:* `scan_001.pdf`
