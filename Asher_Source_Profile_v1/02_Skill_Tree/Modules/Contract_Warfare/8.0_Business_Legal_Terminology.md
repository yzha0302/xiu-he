# Business Legal Terminology Guide (商业法律术语指南)

> **Tags:** `[Terminology]`, `[Equity]`, `[Contracts]`, `[Employment]`, `[Business]`
> **Date Added:** `2026-01-10`
> **一句话总结 (One-Liner):** 掌握商业法律术语是理解合同、保护权益、进行谈判的基础——不懂术语，就像在战场上听不懂敌方的通讯频道。
> **关键协议 (Critical Protocol):**
>
> 1. **术语是武器:** 懂得越多，谈判时越不容易被忽悠
> 2. **中英双语:** 跨境业务必须掌握英文原词
> 3. **场景化理解:** 每个术语都要知道在什么场景下出现

# Part 1: 股权与公司治理 (Equity & Corporate Governance)

## 1.1 股权基础术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Equity** | 股权/权益 | 公司所有权的份额，持有股权=拥有公司的一部分 | "我持有公司10%的股权" |
| **Share/Stock** | 股份/股票 | 股权的具体单位，公司把股权分成N份，每份就是一股 | "公司有100万股，我持有10万股" |
| **Shareholder** | 股东 | 持有公司股份的人或机构 | "股东大会每年召开一次" |
| **Stakeholder** | 利益相关方 | 与公司有利害关系的人，包括股东、员工、客户、供应商等 | "我们需要考虑所有stakeholders的利益" |
| **Authorized Capital** | 授权资本/股本 | 公司章程允许发行的最大股份数量 | "公司授权资本1000万股" |
| **Issued Capital** | 已发行股本 | 公司实际已发行的股份数量 | "已发行500万股" |
| **Paid-up Capital** | 实缴资本 | 股东实际已缴纳的资金 | "注册资本100万，实缴50万" |
| **Par Value** | 面值/票面价值 | 股票的原始额定价值 | "每股面值1元" |
| **Market Value** | 市值 | 公司股票的市场总价值 = 股价 × 总股数 | "公司市值10亿美元" |
| **Book Value** | 账面价值 | 公司资产减去负债的净值 | "账面价值远低于市值" |

## 1.2 股权类型与结构

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Common Stock** | 普通股 | 最常见的股权类型，有投票权，风险和收益都较高 | "创始人持有普通股" |
| **Preferred Stock** | 优先股 | 在分红和清算时优先于普通股，但通常无投票权 | "VC投资通常拿优先股" |
| **Convertible Preferred** | 可转换优先股 | 可以转换成普通股的优先股 | "投资人可以在IPO时转换" |
| **Class A/B Shares** | A/B类股 | 不同类别的股份有不同投票权，常用于创始人控制权设计 | "B类股每股10票，A类股每股1票" |
| **Voting Rights** | 投票权 | 股东在股东大会上投票的权利 | "持股10%但投票权30%" |
| **Vesting** | 归属/成熟 | 股权逐渐解锁生效的机制 | "4年期vesting，每年归属25%" |
| **Cliff** | 悬崖期 | Vesting开始前的等待期 | "1年cliff，cliff之后才开始归属" |
| **Lock-up Period** | 锁定期 | IPO后股东不能出售股票的期限 | "上市后180天锁定期" |

## 1.3 股权变动术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Dilution** | 稀释 | 新发行股份导致现有股东持股比例下降 | "融资后我的股份从20%稀释到15%" |
| **Anti-dilution** | 反稀释 | 保护投资人免受稀释的条款 | "Full ratchet反稀释条款" |
| **Pre-emption Right** | 优先认购权 | 现有股东优先购买新发行股份的权利 | "有权按比例认购新股" |
| **Right of First Refusal (ROFR)** | 优先购买权 | 当股东想卖股时，其他股东优先购买的权利 | "你想卖股，必须先问我要不要买" |
| **Tag-along Right** | 跟售权/共同出售权 | 大股东卖股时，小股东有权跟着一起卖 | "你卖股，我可以一起卖" |
| **Drag-along Right** | 拖售权/强制出售权 | 大股东卖股时，可以强制小股东一起卖 | "我卖股，你必须一起卖" |
| **Put Option** | 卖出期权 | 在约定条件下可以要求公司回购股份 | "5年后我可以让公司回购我的股份" |
| **Call Option** | 买入期权 | 在约定条件下可以购买更多股份 | "业绩达标后我可以加买10%股份" |

## 1.4 公司治理术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Board of Directors** | 董事会 | 公司的决策机构，由股东选举产生 | "董事会每季度开一次会" |
| **Shareholder Meeting** | 股东大会 | 全体股东参加的最高权力机构 | "年度股东大会AGM" |
| **Chairman** | 董事长 | 董事会主席 | "CEO和Chairman可以是同一人" |
| **CEO** | 首席执行官 | 公司日常经营的最高负责人 | "CEO向董事会汇报" |
| **CFO** | 首席财务官 | 负责公司财务的高管 | "CFO管理资金和财务报告" |
| **COO** | 首席运营官 | 负责公司运营的高管 | "COO管理日常业务" |
| **Quorum** | 法定人数 | 会议有效所需的最低出席人数 | "需要过半股东出席才能开会" |
| **Resolution** | 决议 | 股东大会或董事会的正式决定 | "通过董事会决议" |
| **Fiduciary Duty** | 信托责任 | 董事和高管对公司和股东的忠诚义务 | "董事必须为公司利益服务" |

# Part 2: 合同与协议 (Contracts & Agreements)

## 2.1 合同基础术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Contract** | 合同 | 有法律约束力的协议 | "双方签订服务合同" |
| **Agreement** | 协议 | 双方达成的共识，可能有法律效力也可能没有 | "框架协议" |
| **MOU/Memorandum of Understanding** | 谅解备忘录 | 初步意向协议，通常无约束力 | "签MOU表示有意向合作" |
| **LOI/Letter of Intent** | 意向书 | 表明交易意向的文件 | "收购意向书" |
| **Term Sheet** | 条款清单 | 列出主要交易条款的文件 | "投资term sheet" |
| **Definitive Agreement** | 最终协议 | 有完整法律效力的正式协议 | "签署最终收购协议" |
| **Party** | 当事方 | 合同的签约方 | "甲方和乙方" |
| **Consideration** | 对价 | 合同双方交换的价值（英美法要求） | "服务换取报酬" |
| **Execution** | 签署/执行 | 签订合同，或履行合同义务 | "合同已经executed" |
| **Effective Date** | 生效日期 | 合同开始生效的日期 | "自签署之日起生效" |

## 2.2 合同条款术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Recitals** | 序言/鉴于条款 | 合同开头描述背景的部分 | "鉴于甲方拥有... 乙方希望..." |
| **Definitions** | 定义条款 | 解释合同中专业术语的条款 | "'产品'指本合同附件所列产品" |
| **Scope** | 范围 | 合同涵盖的服务或产品范围 | "服务范围包括..." |
| **Term** | 期限 | 合同的有效期限 | "合同期限为2年" |
| **Termination** | 终止 | 结束合同的条件和程序 | "提前30天书面通知可终止" |
| **Renewal** | 续期 | 合同到期后继续的机制 | "自动续期1年" |
| **Amendment** | 修订 | 对合同的修改 | "需双方书面同意方可修订" |
| **Representations & Warranties (R&W)** | 陈述与保证 | 对事实和状态的承诺 | "卖方保证产品无缺陷" |
| **Covenants** | 承诺/约定 | 承诺做某事或不做某事 | "买方承诺不与竞争对手合作" |
| **Indemnification** | 赔偿/补偿 | 一方保护另一方免受损失 | "卖方应赔偿买方因侵权造成的损失" |

## 2.3 责任与赔偿术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Liability** | 责任 | 法律上需要承担的责任 | "卖方应对产品质量承担责任" |
| **Limitation of Liability** | 责任限制 | 限定责任上限的条款 | "赔偿上限为合同总额" |
| **Consequential Damages** | 间接损失 | 由违约间接导致的损失 | "利润损失属于间接损失" |
| **Liquidated Damages** | 约定违约金 | 合同中预先约定的违约赔偿金额 | "延迟一天罚款1000元" |
| **Punitive Damages** | 惩罚性赔偿 | 超出实际损失的惩罚性赔偿（主要美国） | "恶意欺诈可判惩罚性赔偿" |
| **Cap** | 上限 | 责任或赔偿的最高限额 | "责任上限为100万" |
| **Floor** | 下限 | 最低限额 | "赔偿下限为1万" |
| **Basket/Deductible** | 起赔额 | 损失超过此金额才开始赔偿 | "损失超过10万才赔偿" |
| **Gross Negligence** | 重大过失 | 严重疏忽注意义务 | "重大过失不受责任限制" |
| **Willful Misconduct** | 故意不当行为 | 明知故犯 | "故意不当行为无免责" |

## 2.4 争议解决术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Governing Law** | 适用法律 | 用来解释合同的法律 | "本合同适用中国法" |
| **Jurisdiction** | 管辖权 | 有权审理争议的法院 | "争议由北京法院管辖" |
| **Arbitration** | 仲裁 | 由仲裁员裁决争议 | "提交SIAC仲裁" |
| **Mediation** | 调解 | 由第三方促成和解 | "先调解，调解不成再仲裁" |
| **Binding** | 有约束力的 | 裁决或协议具有法律效力 | "仲裁裁决具有binding效力" |
| **Non-binding** | 无约束力的 | 不具有强制执行力 | "调解意见non-binding" |
| **Award** | 裁决 | 仲裁的最终决定 | "仲裁员作出裁决" |
| **Judgment** | 判决 | 法院的最终决定 | "法院判决被告赔偿" |
| **Enforcement** | 执行 | 强制执行裁决或判决 | "申请法院强制执行" |
| **Injunction** | 禁令 | 法院命令禁止某行为 | "法院发出临时禁令" |

# Part 3: 雇佣与劳动 (Employment & Labor)

## 3.1 雇佣关系术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Employer** | 雇主 | 提供工作机会并支付工资的一方 | "公司是员工的雇主" |
| **Employee** | 雇员/员工 | 受雇工作的人 | "公司有100名员工" |
| **Independent Contractor** | 独立承包商 | 非雇员的服务提供者 | "设计师是independent contractor" |
| **Consultant** | 顾问 | 提供专业建议的外部人员 | "聘请法律顾问" |
| **Full-time** | 全职 | 按正常工时工作 | "全职员工每周40小时" |
| **Part-time** | 兼职 | 少于正常工时 | "兼职每周20小时" |
| **Probation** | 试用期 | 正式雇佣前的考察期 | "试用期3个月" |
| **At-Will Employment** | 随意雇佣 | 雇主可无理由解雇（美国特有） | "美国大多数州是at-will" |
| **Fixed-term Contract** | 固定期限合同 | 有明确终止日期的雇佣合同 | "签订2年固定期限合同" |
| **Indefinite Contract** | 无固定期限合同 | 没有终止日期的长期合同 | "连续两次固定期限后转无固定期限" |

## 3.2 薪酬福利术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Salary** | 工资/薪水 | 固定的定期报酬 | "月薪3万" |
| **Wage** | 工资（时薪） | 按小时计算的报酬 | "时薪50元" |
| **Compensation** | 薪酬 | 包括工资和所有福利的总报酬 | "total compensation包括股票期权" |
| **Base Salary** | 基本工资 | 不含奖金和福利的固定工资 | "base salary 20万/年" |
| **Bonus** | 奖金 | 额外的激励报酬 | "年终奖金3个月工资" |
| **Commission** | 佣金/提成 | 按业绩计算的报酬 | "销售提成10%" |
| **Stock Options (ESOP)** | 股票期权 | 以约定价格购买公司股票的权利 | "授予10万股期权" |
| **RSU (Restricted Stock Units)** | 限制性股票单位 | 满足条件后获得的股票 | "每年vest 25% RSU" |
| **Benefits** | 福利 | 非现金形式的报酬 | "医疗保险、带薪假期" |
| **Severance** | 遣散费/离职补偿 | 解雇时支付的补偿 | "N+1遣散费" |

## 3.3 解雇与离职术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Resignation** | 辞职 | 员工主动离职 | "递交辞呈" |
| **Termination** | 解雇/终止 | 雇主结束雇佣关系 | "公司决定terminate该员工" |
| **Dismissal** | 解雇 | 因过错被开除 | "因严重违纪被dismissed" |
| **Redundancy** | 裁员 | 因业务需要减少岗位 | "经济性裁员" |
| **Layoff** | 裁员 | 暂时或永久解雇 | "公司layoff了50人" |
| **Constructive Dismissal** | 推定解雇 | 雇主行为迫使员工辞职 | "降薪50%构成constructive dismissal" |
| **Wrongful Termination** | 不当解雇 | 违法的解雇 | "歧视性解雇属于wrongful termination" |
| **Notice Period** | 通知期 | 终止前需提前通知的期限 | "提前30天通知" |
| **Garden Leave** | 花园假 | 离职前带薪在家 | "通知期内在家休garden leave" |
| **N+1** | N+1补偿 | 中国劳动法规定的解雇补偿 | "工作3年，补偿4个月" |

## 3.4 限制性条款术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Non-Compete Clause (NCC)** | 竞业限制条款 | 离职后不得从事竞争业务 | "2年内不能加入竞争对手" |
| **Non-Solicitation** | 禁止招揽 | 不得招揽原公司员工或客户 | "不能挖原公司的人" |
| **Non-Disclosure (NDA)** | 保密协议 | 不得泄露机密信息 | "签署保密协议" |
| **Confidentiality** | 保密义务 | 对机密信息的保护义务 | "商业秘密有保密义务" |
| **Trade Secret** | 商业秘密 | 有商业价值的机密信息 | "客户名单是trade secret" |
| **IP Assignment** | 知识产权归属 | 工作成果的知识产权归属约定 | "工作期间创造的IP归公司" |
| **Work for Hire** | 职务作品 | 工作期间创造的作品归雇主 | "程序员写的代码是work for hire" |
| **Invention Assignment** | 发明转让 | 发明权归属的约定 | "工作相关发明归公司所有" |

# Part 4: 商业交易 (Business Transactions)

## 4.1 投资融资术语

| 英文术语 | 中文 | 定义 | 例子 |
| :--- | :--- | :--- | :--- |
| **Angel Investment** | 天使投资 | 最早期的个人投资 | "天使轮融资100万" |
| **Seed Round** | 种子轮 | 产品开发阶段的早期融资 | "种子轮500万" |
| **Series A/B/C** | A/B/C轮 | 不同阶段的风险投资轮次 | "完成B轮融资" |
| **Venture Capital (VC)** | 风险投资 | 投资于高增长潜力公司 | "VC投资了这家创业公司" |
| **Private Equity (PE)** | 私募股权 | 投资于成熟公司 | "PE收购了这家公司" |
| **IPO** | 首次公开募股 | 公司首次在证券交易所上市 | "公司计划明年IPO" |
| **Valuation** | 估值 | 公司的价值评估 | "投前估值1亿美元" |
| **Pre-money/Post-money** | 投前/投后估值 | 投资前后的估值 | "投前1亿，投后1.2亿" |

## 4.2 并购术语

# Part 5: Anti-Patterns (反模式)

## 陷阱 1: 混淆Shareholder和Stakeholder

* **Trap:** 认为Shareholder和Stakeholder是一个意思
* **Why:** 发音相似，容易混淆
* **Fix:** Shareholder=股东（持股人），Stakeholder=利益相关方（所有相关人）
* **Example:** "我们要考虑所有stakeholders"——包括员工、客户、供应商，不只是股东

## 陷阱 2: 不理解Vesting

* **Trap:** 以为签了期权协议就马上拥有股票
* **Why:** 不了解vesting机制
* **Fix:** Vesting是逐渐解锁，通常4年期+1年cliff
* **Example:** 10万股期权4年vesting，1年cliff——第一年一股都拿不到，第二年开始才逐渐归属

## 陷阱 3: 混淆Termination和Resignation

* **Trap:** 在英文合同中误用termination描述辞职
* **Why:** 中文都翻译成"离职"
* **Fix:** Resignation=主动辞职，Termination=被动解雇
* **Example:** "I resigned"我辞职了 vs "I was terminated"我被开除了

## 陷阱 4: 忽视Non-Compete的可执行性

* **Trap:** 以为签了竞业限制就一定有效
* **Why:** 不了解各国法律差异
* **Fix:** 美国加州竞业限制基本无效，中国需要支付补偿金才有效
* **Example:** 在加州签的non-compete几乎100%无法执行

## Glossary (术语表)

* **Equity:** 股权——公司所有权的份额
* **Vesting:** 归属——股权逐渐解锁的机制
* **Dilution:** 稀释——新发股导致现有股东比例下降
* **Due Diligence:** 尽职调查——交易前的全面调查
* **Term Sheet:** 条款清单——列出主要交易条款的文件
* **Severance:** 遣散费——解雇补偿
* **Non-Compete:** 竞业限制——离职后不得从事竞争业务
* **M&A:** 并购——公司合并或收购
