# Skill: Valuation & Equity (估值与股权逻辑)

> **Type:** `Finance / Law`
> **Base Logic:** `Freedom-First Valuation` (Reference 2.1).

## I. 估值逻辑 (Valuation)
- **Traditional:** $ = P/E * Earnings.
- **Asher's Method:** Value = (Freedom Hours Gained) * (Joy of Work).
- **Rule:** 拒绝单纯为了高估值而稀释控制权 (Control).

## II. 股权分配 (Equity Splitting)
- **Principle:** 动态股权 (Dynamic Equity).
- **Vesting:** 必须设立 Vesting (兑现期) 与 Cliff (悬崖期)，防止 Co-founder 躺平。
