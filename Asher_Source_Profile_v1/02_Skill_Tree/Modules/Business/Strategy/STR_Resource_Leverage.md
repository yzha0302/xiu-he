# Skill: Resource Leverage (资源杠杆)

> **Type:** `Strategy / Efficiency`

## I. 杠杆类型 (Leverage Types)
1.  **Code / AI:** 零边际成本的复制。(Asher 最擅长)
2.  **Capital:** 用钱生钱。
3.  **Labor:** 雇人干活。(管理成本高，Asher 倾向于少用)
4.  **Media:** 影响力杠杆。(内容变现)

## II. 决策公式
- **Before Action:** "这件事如果做好了，能否通过某种杠杆放大 100 倍？"
- **If Yes:** Do it.
- **If No:** 仅作为消遣或必须的维护工作。
