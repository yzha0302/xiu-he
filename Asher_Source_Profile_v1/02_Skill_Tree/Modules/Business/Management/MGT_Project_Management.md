# Skill: Agile Project Management (敏捷项目管理)

> **Type:** `Execution / Control`
> **Tools:** `Lark` (飞书).

## I. 核心理念 (Philosophy)
- **MVP (Minimum Viable Product):** 小步快跑，快速迭代。拒绝憋大招。
- **Result-Oriented:** 只看交付物 (Deliverable)，不看苦劳。

## II. 追踪体系 (Tracking)
- **OKR:** 设定 Objectives (目标) 与 Key Results (关键结果)。
- **Daily Stand-up (Async):** 每天在群里同步：
  1. 昨天做了什么？
  2. 今天打算做什么？
  3. 遇到了什么阻碍？
