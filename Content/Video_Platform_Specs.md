## 🚀 极速配置 (Universal Best Practices)

如果你不想针对每个平台单独导出，可以使用以下**通用高质量配置**，适用于绝大多数场景：

### 📱 短视频通用配置 (Short Form / Vertical)
*适用：抖音、TikTok、快手、小红书、视频号、Youtube Shorts、Reels*

| 参数 | 推荐值 | 备注 |
| :--- | :--- | :--- |
| **分辨率** | **1080x1920 (1080P)** | 4K 在手机端提升不明显且增加上传难度 |
| **比例** | **9:16** | 竖屏全屏体验最佳 |
| **帧率** | **30 fps** (或 60 fps) | 30fps 最通用，60fps 更流畅但文件更大 |
| **码率 (Bitrate)** | **10 - 15 Mbps** (VBR 1 pass) | 高于此值会被平台二次压缩 |
| **编码** | **H.264 (AVC)** | 兼容性最好 |
| **色彩空间** | Rec.709 (SDR) | HDR 目前各平台兼容性不一，建议暂用 SDR |

### 🎞️ 长视频通用配置 (Long Form / Horizontal)
*适用：B站、YouTube、西瓜视频、视频号(横屏)*

| 参数 | 推荐值 | 备注 |
| :--- | :--- | :--- |
| **分辨率** | **3840x2160 (4K)** | 即使素材是 1080P，导出 4K 也可触发平台高码率编码 |
| **比例** | **16:9** | 传统宽屏 |
| **帧率** | **30 fps** (或 60 fps) | 60fps 适合游戏/运动类 |
| **码率 (Bitrate)** | **20 - 40 Mbps** | 4K 建议 40Mbps+ |

## 🇨🇳 国内平台详细参数

### 1. 🟢 微信视频号 (WeChat Channels)
*互动率最高的黄金比例是竖屏。*

| 项目 | 参数建议 | 注意事项 |
| :--- | :--- | :--- |
| **最佳分辨率** | 1080P (1080x1920) | 4K 支持度有限，1080P 足够 |
| **最佳比例** | **9:16** (竖屏) 或 6:7 | 16:9 (横屏) 会在动态流中显示较小 |
| **时长限制** | 1小时以内 | 推荐 15-60秒 完播率最高 |
| **文件大小** | MAX 2GB | |
| **封面图** | 6:7 或 16:9 | 无论视频比例如何，封面图裁剪要注意 |

### 2. 📺 Bilibili (B站)
*追求画质的首选，支持高帧率高码率。*

| 项目 | 参数建议 | 注意事项 |
| :--- | :--- | :--- |
| **最佳分辨率** | **4K (3840x2160)** | 即使是 1080拍摄，也建议插值导出 4K |
| **最佳比例** | **16:9** (横屏) | 竖屏(9:16) 会进入"Story模式"，流量池不同 |
| **码率** | 推荐 **20Mbps+** | 大会员支持更高码率 |
| **音频** | 320kbps / 48kHz | 音质很重要 |
| **特殊** | 杜比视界/HDR | 支持，需正确设置色域 |

### 3. 📕 小红书 (Xiaohongshu)
*对画质和色彩要求极高，"封面党"必争之地。*

| 项目 | 参数建议 | 注意事项 |
| :--- | :--- | :--- |
| **最佳分辨率** | 1080P (1080x1440 最佳) | 建议导出 4K 以上传最清晰画质 |
| **最佳比例** | **3:4** (1080x1440) | **最推荐**，占据屏幕空间比 9:16 小，但比 16:9 大，适合图文混排流 |
| **备选比例** | 9:16 (1080x1920) | 适合沉浸式全屏视频 |
| **编码** | H.264 | 必须，H.265 可能导致色偏或无法上传 |
| **时长** | < 15分钟 | 15分钟以上需申请权限 |

### 4. 🎵 抖音 (Douyin)
*快节奏，注重开头无需超高码率。*

| 项目 | 参数建议 | 注意事项 |
| :--- | :--- | :--- |
| **最佳分辨率** | 1080P (1080x1920) | 支持 4K，但 1080P 足够 |
| **最佳比例** | **9:16** (竖屏) | 绝对主流 |
| **上传技巧** | 网页端上传 | **网页端上传画质通常优于手机端** |
| **帧率** | 60 fps | 支持高帧率，画面更顺滑 |

## 🌍 Global Platforms (国际平台)

### 5. ▶️ YouTube (油管)
*画质天花板，支持 8K。*

| Form | Specs | Note |
| :--- | :--- | :--- |
| **Long Form** | **4K (3840x2160) / 16:9** | Always upscale to 4K to force VP9 codec for better 1080p quality |
| **Vertical (Shorts)** | **1080x1920 / 9:16** | Duration must be < 60 seconds |
| **Frame Rate** | 24, 30, 60 fps | Match your source footage |

### 6. 📸 Instagram (Reels & Posts)
*Mobile-first, compression is aggressive.*

| Type | Specs | Note |
| :--- | :--- | :--- |
| **Reels** | **1080x1920 (9:16)** | **Most Important**. Do not put text near edges (Safe Zone). |
| **Post (Feed)** | **1080x1350 (4:5)** | Only strictly vertical 4:5, 16:9 looks tiny. |
| **Bitrate** | 3.5 Mbps - 5 Mbps | Don't export too high, IG will crush it. |
| **Upload** | Use "High Quality Upload" | Check settings in mobile app: Settings > Account > Data Usage. |
